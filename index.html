📖 PUBLISH GUIDE

🚀 GitHub Pages で無料公開する方法

1

GitHubアカウントを作る

github.com にアクセスして「Sign up」から登録。ユーザー名がそのままURLになります（例：tanaka → tanaka.github.io）

2

リポジトリを作る

「New repository」をクリック。名前を ユーザー名.github.io にする（これが重要！）。「Public」を選んで作成。

3

ファイルをアップロード

「uploading an existing file」をクリックして index.html と games/ フォルダをドラッグ＆ドロップ。「Commit changes」で保存。

4

GitHub Pages をオンにする

リポジトリの「Settings」→「Pages」→ Source を main ブランチに設定して保存。数分後に公開完了！

5

ゲームを追加するとき

新しいゲームのHTMLファイルを games/ フォルダにアップロードして、index.html のゲームカードをコピペして増やすだけ！
